fun <AllT : Any, EachT, ValueT> OptionWithValues<AllT, EachT, ValueT>.validate(validator: OptionValidator<AllT>): OptionDelegate<AllT>

Check the final option value and raise an error if it's not valid.

The validator is called with the final option type (the output of transformAll), and should call fail if the value is not valid. It is not called if the delegate value is null.

You can also call require to fail automatically if an expression is false.

Example:

val opt by option().int().validate { require(it % 2 == 0) { "value must be even" } }
