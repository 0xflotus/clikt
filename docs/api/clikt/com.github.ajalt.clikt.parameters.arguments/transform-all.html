transformAll

fun <AllInT, ValueT, AllOutT> ProcessedArgument<AllInT, ValueT>.transformAll(nvalues: Int? = null, required: Boolean? = null, transform: ArgCallsTransformer<AllOutT, ValueT>): ProcessedArgument<AllOutT, ValueT>

Transform all values to the final argument type.

The input is a list of values, one for each value on the command line. The values in the list are the output of calls to convert. The input list will have a size of nvalues if nvalues is > 0.

Used to implement functions like paired and multiple.

Parameters

nvalues - The number of values required by this argument. A negative nvalues indicates a variable number of values.

required - If true, an error with be thrown if no values are provided to this argument.
